Detailed Description

A coordinate descent algorithm for multi-class support vector machines by Crammer and Singer

min_{} 0.5 ||w_m()||^2 + e^m_i alpha^m_i s.t. ^m_i <= C^m_i m,i , ^m_i=0 i

where e^m_i = 0 if y_i = m , e^m_i = 1 if y_i != m , C^m_i = C if m = y_i , C^m_i = 0 if m != y_i, and w_m() = ^m_i x_i

Given: x, y, C eps is the stopping tolerance

solution will be put in w

See Appendix of LIBLINEAR paper, Fan et al. (2008)
