A coordinate descent algorithm for L1-loss and L2-loss SVM dual problems

min_ 0.5(^T (Q + D)) - e^T , s.t. 0 <= <= upper_bound_i ,

where Qij = yi yj xi^T xj and D is a diagonal matrix

In L1-SVM case: upper_bound_i = Cp if y_i = 1 , upper_bound_i = Cn if y_i = -1 , D_ii = 0 In L2-SVM case: upper_bound_i = INF , D_ii = 1/(2*Cp) if y_i = 1 , D_ii = 1/(2*Cn) if y_i = -1

Given: x, y, Cp, Cn eps is the stopping tolerance

solution will be put in w

See Algorithm 3 of Hsieh et al., ICML 2008

A coordinate descent algorithm for L1-loss and L2-loss epsilon-SVR dual problem

min_ 0.5^T (Q + diag(lambda)) - p {i=1}^l|| + {i=1}^l yi , s.t. -upper_bound_i <= <= upper_bound_i ,

where Qij = xi^T xj and D is a diagonal matrix

In L1-SVM case: upper_bound_i = C , lambda_i = 0 In L2-SVM case: upper_bound_i = INF , lambda_i = 1/(2*C)

Given: x, y, p, C eps is the stopping tolerance

solution will be put in w

See Algorithm 4 of Ho and Lin, 2012
